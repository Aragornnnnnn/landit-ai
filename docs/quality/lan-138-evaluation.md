# LAN-138 AI 응답 품질 평가

## 평가 기준

- 마무리 멘트는 마지막 사용자 발화와 상대 역할, 구체적인 상황을 반영한다.
- 마무리 멘트는 새 질문이나 대화 종료를 선언하는 메타 표현을 포함하지 않는다.
- `GOOD`은 의미가 명확하고 평가 컨텍스트의 의도를 충족하며 실제 수정이 필요하지 않은 발화다.
- `NEEDS_IMPROVEMENT`는 의미, 문법, 맥락 적합성 또는 역할에 실제 영향을 주는 문제와 구체적 수정 표현이 있을 때만 사용한다.
- 더 세련된 표현이 가능하다는 이유만으로 `NEEDS_IMPROVEMENT`로 판정하지 않는다.

## 수정 전 기준선

- 모델. `openai/gpt-5.4-mini`.
- 사례. `tests/fixtures/lan_138_quality_cases.json`의 비식별화한 8개 사례.
- 반복 횟수. 사례별 3회.
- 결과. OpenRouter API key가 현재 Codex 작업 셸에 주입되지 않아 미실행.
